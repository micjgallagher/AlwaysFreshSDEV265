# Always Fresh

## Project Description
Always Fresh is a smart fridge web application designed to help users track food inventory, reduce food waste, and improve meal planning.

## Features
- Homepage with hero section
- Features page
- About page
- Contact page
- Demo/test navigation
- Capri-themed user interface

## Tools Used
- HTML
- CSS
- GitHub
- GitHub Pages

## Setup Instructions
1. Download or clone the repository.
2. Open the project folder.
3. Make sure all files stay in their correct folders:
   - `index.html`
   - `pages/`
   - `images/`

## Installation
No special installation is required.

## How to Run
1. Open `index.html` in a web browser  
   or  
2. Visit the live GitHub Pages link.
